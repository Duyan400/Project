#Convention de style

L'équipe utilise les conventions suivantes:

Nommage:

Pour le nom des variables, le nom doit être en français et nous suivons le plus possible les normes JAVA 
(http://www.oracle.com/technetwork/java/javase/documentation/codeconvtoc-136057.html)

Pour les noms de Classe:

La première lettre du nom de classe débute par une minuscule, les mots suivants débutent pas une majuscule.
Nom le plus clair possible.
Aucune lettre ou chiffre seul.
Evité les noms dont le sens est difficile à comprendre.

Constantes:

Claires
Mots séparer par un underscore " _ ".

Autre convention a suivre:

Un espace devant chaque {

Un saut de ligne après chaque ouverture d'une {
Un saut de ligne après chaque fermeture d'une }

Tabulation de 4 caractères.
